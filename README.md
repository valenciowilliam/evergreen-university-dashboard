# 🎓 Evergreen University Dashboard

A full-stack web application for managing student registrations using **React, Express, and PostgreSQL (Neon)**.

---

## 🚀 Live Demo

- 🌐 Frontend (Vercel): https://evergreen-university-dashboard.vercel.app/ 
- ⚙️ Backend (Render): https://evergreen-university-dashboard.onrender.com

---

## ✨ Features

- 📝 Student Registration Form  
- ✅ Form Validation  
- 👀 Preview Card before submission  
- 🔗 REST API Integration  
- 🗄️ Data stored in PostgreSQL (Neon)  
- 🔄 Multi-page navigation using React Router  

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- Axios  
- React Router  

### Backend
- Node.js  
- Express.js  

### Database
- Neon PostgreSQL  

### Deployment
- Vercel (Frontend)  
- Render (Backend)
